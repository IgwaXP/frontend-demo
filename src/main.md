## Clone and Setup
[Download]() or clone repository to your local computer.
```
$ git clone 
```

Install
```
npm install
```

Start JSON Server
```
npm run json:server (localhost:3000)
```

Compiles and hot-reloads for development
```
npm run serve (localhost:8080)
```

Compiles and minifies for production
```
npm run build
```
